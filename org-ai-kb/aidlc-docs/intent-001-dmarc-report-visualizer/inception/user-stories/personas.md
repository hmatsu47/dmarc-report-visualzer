# Personas

## メールインフラ管理者（Mail Infrastructure Engineer）

- **名前**: 田中（代表的ペルソナ）
- **役割**: 組織のメールインフラを管理するエンジニア。DMARCポリシーの運用・監視を担当。
- **ゴール**:
  - 自組織ドメインのDMARC認証状況を把握し、なりすましメールの有無を検知する
  - SPF/DKIMの設定不備を早期発見し、正規メールの配信率を維持する
  - DMARCポリシーの段階的強化（none → quarantine → reject）の判断材料を得る
- **コンテキスト**:
  - DMARCの基礎知識を持つエンジニア（XML構造・認証フローを理解）
  - 5名程度のチームで共有利用
  - 日常的にはダッシュボードで傾向を確認し、異常時に詳細を調査する
  - CDKデプロイやAWSコンソール操作に習熟している
